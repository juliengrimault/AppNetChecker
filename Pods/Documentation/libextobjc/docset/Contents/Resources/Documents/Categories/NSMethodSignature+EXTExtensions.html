typeEncoding

Returns the Objective-C type encoding for this method signature, which includes the return type and all arguments. The resultant string matches the format of \c method_getTypeEncoding() and is suitable for passing to \c class_addMethod() and similar facilities.

- (const char *)typeEncoding

Discussion

Returns the Objective-C type encoding for this method signature, which includes the return type and all arguments. The resultant string matches the format of \c method_getTypeEncoding() and is suitable for passing to \c class_addMethod() and similar facilities.

Note: The returned string is autoreleased.
